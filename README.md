## Feature Implementation Rules (ম্যানেজার মার্ক ১০ পয়েন্ট)

- **কমপ্লিট ফিচার** = **Frontend + Backend** দুটোই থাকতে হবে।
- ফিচারটি **Problem Identification** এবং **ওয়েব অ্যাপের মাধ্যমে Solution** এর উপর ভিত্তি করে হতে হবে।
- **শুধু পেজ ডিজাইন বা কালার করেকশন** ফিচার হিসেবে কাউন্ট হবে না।
- প্রত্যেক মেম্বারকে **কমপ্লিট ফিচার** অনুযায়ী ম্যানেজার মার্ক দেবেন।

**প্রত্যেক টিম মেম্বারকে প্রজেক্ট জুড়ে কমপ্লিট ও ফুলি ফাংশনাল এমন কমপ্লিট মেজর ফিচার রিসার্চ ও ইমপ্লিমেন্ট করতে হবে — অন্তত ৩টি।**
