# SkillVoyager.AI – Your AI-Powered Learning Journey

**EdTech / Career Development** — AI-powered personalized learning & career roadmap web application.

---


## Tech Stack

- **Frontend:** React/JSX, Tailwind CSS (existing HTML pages as reference)
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **AI:** Gemini API
- **State:** Context API (or similar)

---

## প্রজেক্ট স্ট্রাকচার (সংক্ষেপে)

- `landing/` — ল্যান্ডিং পেজ ও ১০টা সেকশন কম্পোনেন্ট
- `pages/` — অ্যাপ পেজগুলো (auth, dashboard, roadmaps, courses, ইত্যাদি)
- `js/` — থিম টগল ইত্যাদি
- `task-details/` — টিম মেম্বারওয়াইজ টাস্ক ডিটেইলস (মার্ক ডিস্ট্রিবিউশন ও লিংক এই রিডমিতে কানেক্টেড)
